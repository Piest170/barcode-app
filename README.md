## เทคโนโลยี
- **Backend**: C# .NET 9 Web API + EF Core + SQLite
- **Frontend**: Vue 3 + Vite + Bootstrap 5
- **Features**: เพิ่ม/ลบ/แสดง barcode/validate

## วิธีเริ่มต้นใช้งาน
### Backend
- cd backend/ProductCodeApi
- dotnet run
### Run Test
- cd ../ProductCodeApi.Tests
- dotnet test
### Frontend
- cd frontend
- npm install
- npm run dev
